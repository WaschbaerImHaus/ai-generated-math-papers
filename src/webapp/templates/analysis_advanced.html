∫ Erweiterte Analysis

Maßtheorie (Lebesgue-Integral, σ-Algebren), Funktionalanalysis (Banach, Hilbert, Fixpunktsatz), Partielle Differentialgleichungen, Operatoralgebren (C*-, von-Neumann-) und Spezielle Funktionen (Bessel, Legendre, Gamma).

μ Maßtheorie

Ein Maßraum $(\Omega, \mathcal{F}, \mu)$ besteht aus einer Menge $\Omega$, einer $\sigma$-Algebra $\mathcal{F}$ und einem Maß $\mu: \mathcal{F} \to [0,\infty]$. Das Lebesgue-Integral $\int_a^b f \, d\mu$ ist für Funktionen definiert, die bzgl. $\mu$ messbar sind. Konvergenzsätze: Monotone Konvergenz (Beppo Levi), Dominierte Konvergenz (Lebesgue), Satz von Fubini.

Lebesgue-Integral $\int_a^b f \, d\lambda$

Das Lebesgue-Integral ist für eine viel größere Klasse von Funktionen definiert als das Riemann-Integral. Für Riemann-integrierbare Funktionen stimmen beide überein. Schritt-/Treppenfunktionen werden exakt integriert.

$\sigma$-Algebra aus Generatoren

$\sigma(\mathcal{G})$: kleinste $\sigma$-Algebra, die $\mathcal{G}$ enthält. Abgeschlossen unter Komplementen, abzählbaren Vereinigungen und Schnitten. Die Borel-$\sigma$-Algebra $\mathcal{B}(\mathbb{R}) = \sigma(\text{offene Mengen})$.

Konvergenzsätze der Maßtheorie

Monotone Konvergenz (Beppo Levi): $0 \leq f_n \nearrow f$ $\Rightarrow$ $\int f_n \to \int f$.
Dominierte Konvergenz (Lebesgue): $|f_n| \leq g \in L^1$, $f_n \to f$ $\Rightarrow$ $\int f_n \to \int f$.
Fubini: $\int\int f \, d\mu \, d\nu = \int\int f \, d\nu \, d\mu$.

‖ Funktionalanalysis

Ein Banach-Raum ist ein vollständiger normierter Vektorraum. Ein Hilbert-Raum ist ein vollständiger Vektorraum mit Skalarprodukt. Vier große Sätze: Hahn-Banach, offene Abbildung, abgeschlossener Graph, gleichmäßige Beschränktheit. Rieszscher Darstellungssatz: Jedes stetige Funktional auf $H$ ist von der Form $\langle \cdot, y \rangle$.

Banach-Raum $L^p$

$\|f\|_{L^p} = \left(\int |f|^p \, d\mu\right)^{1/p}$. $L^1$: absolut integrierbare Funktionen. $L^2$: quadratisch integrierbar (Hilbert-Raum). $L^\infty$: essentiell beschränkte Funktionen. Dreiecksungleichung (Minkowski): $\|u+v\|_p \leq \|u\|_p + \|v\|_p$.

Hilbert-Raum $L^2$

Skalarprodukt $\langle u, v \rangle = \int u \overline{v} \, d\mu$. Cauchy-Schwarz: $|\langle u,v \rangle| \leq \|u\| \cdot \|v\|$. Parseval: $\sum_n |\langle f, e_n\rangle|^2 = \|f\|^2$. Riesz-Fischer: $L^2$ ist vollständig.

Banachscher Fixpunktsatz

Sei $f: X \to X$ eine Kontraktion auf einem vollständigen metrischen Raum: $d(f(x), f(y)) \leq L \cdot d(x,y)$, $0 \leq L < 1$. Dann hat $f$ einen eindeutigen Fixpunkt: $\exists! x^* : f(x^*) = x^*$. Fehlerabschätzung: $d(x_n, x^*) \leq \frac{L^n}{1-L} d(x_0, x_1)$.

∂ Partielle Differentialgleichungen

Klassifikation linearer PDE 2. Ordnung $A u_{xx} + B u_{xy} + C u_{yy} + \ldots = 0$: Elliptisch ($B^2 - 4AC < 0$, z.B. Laplace $\Delta u = 0$), Parabolisch ($B^2 - 4AC = 0$, z.B. Wärmeleitung $u_t = \alpha u_{xx}$), Hyperbolisch ($B^2 - 4AC > 0$, z.B. Welle $u_{tt} = c^2 u_{xx}$).

PDE klassifizieren

Gegeben $Au_{xx} + Bu_{xy} + Cu_{yy} + \ldots = 0$: Diskriminante $D = B^2 - 4AC$ bestimmt den Typ. Beispiele: Laplace $(A=C=1, B=0)$, Wärme $(A=1, B=C=0)$, Welle $(A=1, C=-1, B=0)$.

Wärmeleitungsgleichung $u_t = \alpha u_{xx}$

Numerische Lösung: Explizites Finite-Differenzen-Verfahren. Analytische Lösung: Fourier-Reihe $u(x,t) = \sum_n c_n e^{-\alpha n^2\pi^2 t/L^2} \sin(n\pi x/L)$. Stabilitätsbedingung: $\alpha \Delta t / \Delta x^2 \leq 1/2$.

Wellengleichung $u_{tt} = c^2 u_{xx}$

D'Alembert-Lösung: $u(x,t) = \frac{1}{2}[f(x-ct) + f(x+ct)] + \frac{1}{2c}\int_{x-ct}^{x+ct} g(s)\,ds$. Numerische Lösung mit explizitem Schema (CFL-Bedingung: $c\Delta t/\Delta x \leq 1$).

⊙ Operatoralgebren

Eine C*-Algebra ist eine Banach-*-Algebra mit $\|a^*a\| = \|a\|^2$. Gelfand-Transform: Jede kommutative C*-Algebra ist isomorph zu $C(X)$ für einen kompakten Hausdorff-Raum $X$. von-Neumann-Algebren: schwach abgeschlossene *-Unteralgebren von $B(H)$. Faktoren: Typ I (diskret), II (kontinuierlich), III (typisch). K-Theorie: Homologietheorie für C*-Algebren; $K_0, K_1$.

C*-Algebren: Gelfand & GNS

GNS-Konstruktion: Jede C*-Algebra ist isometrisch isomorph zu einer Operatoralgebra auf einem Hilbert-Raum. Cuntz-Algebren $\mathcal{O}_n$: Einfache, rein unendliche C*-Algebren, erzeugt von $n$ Isometrien.

von-Neumann-Algebra Faktoren

Typ I: $B(H)$, diskrete Quantensysteme.
Typ II₁: endliche Faktoren mit normierter Spur (z.B. $L^\infty(\mathbb{R}) \rtimes \mathbb{R}$).
Typ III: Faktoren ohne Spur, Quantenfeldtheorie. Murray-von-Neumann-Klassifikation 1936–43.

K-Theorie für C*-Algebren

$K_0(A)$: Grothendieck-Gruppe der stabilen Äquivalenzklassen von Projektionen. $K_1(A)$: Gruppe der Invertiblen in Matrizen über $A$ (modulo Homotopie). Bott-Periodizität: $K_n \cong K_{n+2}$. Atiyah-Singer-Indexsatz.

ɸ Spezielle Funktionen

Spezielle Funktionen treten als Lösungen wichtiger DGLs auf. Bessel-Funktionen $J_n(x)$: Zylindrische Symmetrie (Elektrodynamik, Akustik). Legendre-Polynome $P_n(x)$: Kugelflächenfunktionen (Quantenmechanik). Gamma-Funktion: $\Gamma(z+1) = z\Gamma(z)$, Verallgemeinerung der Fakultät.

Bessel-Funktionen $J_n(x)$

Bessel-DGL: $x^2 y'' + xy' + (x^2 - n^2)y = 0$. Lösungen: $J_n(x) = \sum_{m=0}^\infty \frac{(-1)^m}{m!\Gamma(m+n+1)} \left(\frac{x}{2}\right)^{2m+n}$. Orthogonalität auf $[0, a]$ bzgl. $x$.

Legendre-Polynome $P_n(x)$

Legendre-DGL: $(1-x^2)y'' - 2xy' + n(n+1)y = 0$. Rodrigues-Formel: $P_n(x) = \frac{1}{2^n n!} \frac{d^n}{dx^n}(x^2-1)^n$. Orthogonalität: $\int_{-1}^1 P_m P_n \, dx = \frac{2}{2n+1}\delta_{mn}$.

Gamma-Funktion & Verwandte

$\Gamma(z) = \int_0^\infty t^{z-1} e^{-t} \, dt$ für $\mathrm{Re}(z) > 0$. Spiegelungsformel: $\Gamma(z)\Gamma(1-z) = \frac{\pi}{\sin(\pi z)}$. Legendresches Verdopplungsformel, Gauß-Darstellung. Zusammenhang: $\Gamma(1/2) = \sqrt{\pi}$, $\Gamma(n+1) = n!$.
