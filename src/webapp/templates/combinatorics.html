■ Kombinatorische Abzählung

Binomialkoeffizient: Anzahl der $k$-elementigen Teilmengen einer $n$-Menge: $$\binom{n}{k} = \frac{n!}{k!\,(n-k)!}$$ Partitionszahlen $p(n)$: Anzahl der Zerlegungen von $n$ in positive Summanden (ungeordnet). Eulersche Formel: $\sum_{n\geq 0} p(n)\,x^n = \prod_{k=1}^{\infty} \frac{1}{1-x^k}$. Stirling-Zahlen 2. Art $S(n,k)$: Anzahl der Partitionen einer $n$-Menge in genau $k$ nichtleere Blöcke. Rekursion: $S(n,k) = k\,S(n-1,k) + S(n-1,k-1)$. Motzkin-Zahlen $M_n$: Anzahl der Pfade von $(0,0)$ nach $(n,0)$ mit Schritten $(1,1)$, $(1,0)$, $(1,-1)$, die $y \geq 0$ halten.

◆ Ramsey-Theorie

Ramseys Satz: Für alle $s, t \geq 2$ existiert eine Zahl $R(s,t)$ (Ramsey-Zahl), so dass jede 2-Färbung der Kanten des vollständigen Graphen $K_{R(s,t)}$ einen roten $K_s$ oder einen blauen $K_t$ enthält. $$R(s,t) \leq \binom{s+t-2}{s-1}$$ Taubenschlagprinzip (Dirichlet): Werden $n$ Objekte auf $k < n$ Schachteln verteilt, enthält mindestens eine Schachtel $\geq \lceil n/k \rceil$ Objekte. Wichtige Werte: $R(3,3) = 6$, $R(4,4) = 18$, $R(3,5) = 14$.

∑ Erzeugende Funktionen

Eine Erzeugende Funktion $F(x) = \sum_{n=0}^{\infty} a_n x^n$ kodiert eine Folge $(a_n)$ als formale Potenzreihe. Fibonacci-Zahlen: $F_n = F_{n-1} + F_{n-2}$, $F_0=0$, $F_1=1$. Erzeugende Funktion: $$F(x) = \frac{x}{1 - x - x^2} = \sum_{n=0}^{\infty} F_n\, x^n$$ Catalan-Zahlen: $C_n = \frac{1}{n+1}\binom{2n}{n}$, zählen z.B. korrekte Klammerungen und planare Binärbäume. $$C(x) = \frac{1 - \sqrt{1-4x}}{2x} = \sum_{n=0}^{\infty} C_n\, x^n$$
